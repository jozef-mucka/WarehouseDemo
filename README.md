# WarehouseDemo
Demo SpringBoot Java web app made after watching a Youtube video course and remembering what I used at work some time ago.
If you need to test me further for job interview purposes, write me a message what else I should implement.
Technologies used: Spring Boot, Thymeleaf, jQuery.

After successfully building and launching, pages are available at:
Multi-page, no-JavaScript version with Thymeleaf: http://localhost:8080/products/
Single-page version with jQuery: http://localhost:8080/products/spa

REST API:
http://localhost:8080/api/v1/products/
http://localhost:8080/api/v1/products/{id} - supported GET, POST, PUT, DELETE methods
http://localhost:8080/api/v1/products/search/{searchTerm}
