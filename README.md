# WarehouseDemo
Demo SpringBoot Java web app made after watching a Youtube video course and remembering what I used at work some time ago.<br />
If you need to test me further for job interview purposes, write me a message what else I should implement.<br />
Developed using Spring Tools for Eclipse, Version: 4.32.2<br />
Technologies used: Spring Boot, Thymeleaf, jQuery.

After successfully building and launching, pages are available at: <br />
Multi-page, no-JavaScript version with Thymeleaf: http://localhost:8080/products/ <br />
Single-page version with jQuery: http://localhost:8080/products/spa <br />

REST API: <br />
http://localhost:8080/api/v1/products/ <br />
http://localhost:8080/api/v1/products/{id} - supported GET, POST, PUT, DELETE methods <br />
http://localhost:8080/api/v1/products/search/{searchTerm} <br />
